Basic Terminology

Variables - Variables are assigned values using the = operator. They can any value of different data types like strings, integers and floats

Operators - Operators are symbols like * / + -. They can be used to perform mathematic and arithmetic operations.

Commenting - Commenting can be done by using # to explain what you code does.


Basic Python Data Types


String

  • Strings are sequences of letters surrounded by quotations
  • Strings can be concatenate with other strings
  • stringExample = 'Hello, I am a string'

Integer

  • Integers are numbers without decimals (whole numbers)
  • integerExample = 420

Float

  • Floats are numbers with a decimal
  • floatExample = 420.5

Boolean

  • A variable that is a boolean can either be true or false
  • boolExample = true

If... Else Statments

  • a = 200
b = 33
if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")
else:
    print("a is greater than b")
  • If a condition is true, the action inside the code block is executed
  • If the condition is false, the next else or elif statement is run
  • Else If statements allow you run different actions based on different conditions

For and While loops


While loops

  • i = 1
while i < 6:
print(i)
i += 1
  • While loops allow you to loop until a condition is satisfied

For loops

  • for x in "banana":
print(x)
  • For loops allow you to loop through data or lists to perform actions

Important Built-in Functions

print() - prints the value of the variable or string inside the brackets

.format() - replaces the placeholders that are made from curly brackets

.upper() - change a string to uppercase letters

input() - ask a user for input which then be assigned to variables
